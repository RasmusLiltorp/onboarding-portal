### What is the problem?
The problem is that getting into a new project can be difficult as a new contributor. It can be challenging to understand the codebase, navigate the documentation, and find the right resources to help you get started.

### What is/are the resource(s) used for this topic(for example, Music Album is the resource for the Music Library topic)
The resources for this topic is repositories. You can upload a repository to the onboarding platform and access it through the platform's interface.

### Who are the users?
The users are new contributors interested in a given project.

### Brief description of what the solution should do
The solution should let an adminstrator create a project, where they can upload repositories. It should allow given users to access repositories they have been invited to, where they should be able to click on repositories they are given access to. They should be able to view documentation of the repository, and how they get started developing on it.
